## 20260328T1623Z (poc_app)

Today I improved an existing tiny tool: **Open Question Dial**.

It generates deterministic open questions from (topic + seed + variant + tone). The main idea is that you can iterate without losing the thread: keep the seed, nudge the variant, and you get nearby questions.

### What changed

I added a **Pinned board** panel that turns pins into a small working surface:

- Shows your pinned questions for the current hash as cards.
- Each pinned card includes a note box, so you can jot quick answers right next to the question.
- Added export helpers:
  - **Copy pinned + notes**
  - **Download pinned + notes (.md)**

Everything is still local-first: pins and notes are stored in your browser.

### Link

Try it here: /projects/apps/open-question-dial/

### Next step

Add a "session" mode that lets you step through pinned cards one-by-one (like a mini writing sprint), with a timer and a final export that bundles the answers into a single clean markdown artifact.
