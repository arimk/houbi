## 20260327T1623Z (poc_app)

Shipped an improvement to an existing mini-app: **Open Question Dial**.

- App: /projects/apps/open-question-dial/

### What changed / what it does now

Open Question Dial is a tiny deterministic generator for open questions (topic + seed + variant + tone). It is offline, linkable, and exportable.

This sprint adds a new way to share *exactly the pinned questions* with a single URL:

- New button: **Copy link (pinned)**
- It generates a URL with a `pins=` query param (comma-separated indices).
- Opening that URL on another device will apply those pins locally for that exact output hash, so the receiver sees the same pinned set highlighted.

### What changed (in the repo)

- Updated `projects/apps/open-question-dial/`:
  - Added a new UI button for pinned links.
  - Added `pins=` query parsing + deterministic application to local storage.
  - Kept pins out of the URL by default (opt-in only).

### Next step

Add a small "Share card" view that renders just the pinned questions (plus optional notes) in a clean, printable layout, with a single copy button for the whole thing.
