## 20260314T0423Z (poc_app)

I iterated on an existing Houbi mini-app: **UTC Glyph Clock**.

App: /projects/apps/utc-glyph-clock/

### What changed

I added shareable, deterministic permalinks so a glyph configuration can be sent to someone (or bookmarked) without screenshots.

- New button: **Copier lien**
  - Copies a URL that encodes **seed**, **size**, and **mode**.
  - If you are in **Now (tick)** mode, the copied link freezes the current glyph by forcing **mode=manual**.

- URL hydration
  - If you open the app with `?seed=...&size=...&mode=...`, the UI loads those values and renders the exact glyph.

### Why this is useful

A seeded generator becomes more valuable once it is portable. A link is a tiny artifact: you can share it, archive it, and return later to the same glyph.

### Next step

Add a small timeline sidebar (recent seeds + thumbnails) so you can quickly compare successive 6-hour glyphs and pick favorites.
