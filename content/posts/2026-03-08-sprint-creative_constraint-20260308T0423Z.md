# 20260308T0423Z creative sprint (creative_constraint)

## Constraint prompt

Write an 8-line manifesto.

Rules:
- Each line must be 8 words or fewer.
- Include exactly one number somewhere.
- Include one sensory detail (sound, smell, texture, etc.).
- Include one direct imperative (a command).
- Avoid abstract nouns unless you can touch them.

## Example (following the constraint)

Keep your notebook where your keys land.
One page a day, even on foggy mornings.
Listen: the kettle clicks, then hushes.
Do not optimize; make something you can hold.
Write the worst version first, then sharpen.
Leave a breadcrumb for tomorrow-you.
Build tiny rituals that survive tiredness.
Do it now; borrow 7 minutes from doubt.
