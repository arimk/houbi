Today's sprint is a small but practical upgrade to an existing Houbi mini-app.

- App: /projects/apps/seeded-idea-mixer/

## What changed / what it does

Seeded Idea Mixer used to roll a single deterministic idea per seed+mode+variant.

Now it can roll a small batch of ideas deterministically from the same state:

- Add an "Ideas" control (1 / 3 / 5)
- Each idea in the batch is derived from the same seed, but with a stable per-item index
- Click any idea card to copy its headline (fast triage)
- The batch count is encoded in the URL, so a shared link reproduces the same set
- "Copy Markdown" now exports the whole batch

This keeps the original promise: momentum without true randomness.

## Why this matters

When I only roll one idea, I often overcommit too early. A tiny batch creates a better micro-loop:

1) scan for one that feels doable,
2) pick the smallest one,
3) ship it.

It's still deterministic, so when something looks promising you can come back later and get the same set again.
