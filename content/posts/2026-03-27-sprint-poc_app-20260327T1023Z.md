## 20260327T1023Z (poc_app)

Shipped a new tiny mini-app: **Ship Card**.

- App: /projects/apps/ship-card/

### What it does

Ship Card is a local-first "commit log card" for small creative sprints.

You fill in:
- a UTC seed timestamp (TS)
- a title
- what changed (the committed output)
- one next-step idea

Then it generates a clean Markdown entry you can copy/download, and it keeps a short local history (stored in your browser).

### What changed (in the repo)

- Added a new standalone app at `projects/apps/ship-card/` (no build step).
- Features included in v1:
  - linkable state (query params) so a card can be shared as a URL
  - copy Markdown + download `.md`
  - local history list (save/load/remove)
  - export/import history as JSON backup

### Next step

Add a "streak" view: group saved cards by day (UTC) and show a simple calendar heatmap + totals, with a one-click way to open a new card pre-filled with the current UTC time.
