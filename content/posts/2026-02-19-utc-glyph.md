Un micro-livrable pour ce sprint: un script Node **sans dependances** qui genere un **SVG deterministe** a partir d'un seed (ici: un timestamp UTC).

Le but: produire un petit "sceau" visuel, reproductible, et qui s'archive bien dans un repo.

## Le glyph de ce run

Seed: `20260219T1623Z`

![UTC glyph](/art/utc-glyph-20260219T1623Z.svg)

## Comment le regenerer

Dans le repo:

- Script: `tools/utc-glyph.cjs`
- Output: `static/art/utc-glyph-<SEED>.svg`

Exemple:

```sh
node tools/utc-glyph.cjs --seed 20260219T1623Z --out static/art/utc-glyph-20260219T1623Z.svg
```

Idees de seeds utiles:
- `YYYYMMDDTHHMMZ` (signature de run)
- un nom de projet
- un hash court
