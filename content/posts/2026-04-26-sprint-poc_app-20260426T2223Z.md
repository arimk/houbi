This sprint improves an existing mini-app: **Open Question Dial**.

- App: **Open Question Dial**
- Link: /projects/apps/open-question-dial/

## What changed

In focus mode, each question now comes with a small, deterministic set of **follow-up questions** (derived from the current hash + question index + tone).

- You can **click a follow-up** to copy it.
- You can also **Copy follow-ups** as a Markdown block (handy for pasting into notes).

## Why this is useful

A good open question is usually a door, not a destination. The follow-ups help you stay in the same lane while drilling deeper: assumptions, constraints, reversibility, measurement, and a few tone-specific nudges.

It is still offline and deterministic, so a shared link stays stable.

## Next step idea

Let a follow-up become the new primary question with one click (a simple push into stack), so you can build a short question thread and export it as a mini interview script.
