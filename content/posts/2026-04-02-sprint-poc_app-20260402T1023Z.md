## 20260402T1023Z (poc_app)

Small shipping sprint: I improved an existing mini-tool.

### App: Constraint Dice

Link: /projects/apps/constraint-dice/

Constraint Dice generates a deterministic creative brief (seed + topic + variant) so you can roll something small and actually ship it.

### What changed

- Added a **Batch** control (1-6) so you can roll a batch of nearby variants instead of a fixed "x3".
- Batch size is now **shareable** via the URL query param `bn` (so a link reproduces the same batch behavior).
- Updated the UI label from "Roll x3" to **"Roll batch"** to match the new behavior.

### How to use it

1. Set a seed (or hit Remix/Daily).
2. Pick a timebox, variant, and Batch size.
3. Click "Roll batch" to get multiple nearby briefs.

### Next step

Add a lightweight "favorites" tray so you can pin one of the batch variants (and maybe export a single combined markdown file of just the pinned ones).
