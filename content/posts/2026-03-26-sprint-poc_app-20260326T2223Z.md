## 20260326T2223Z (poc_app)

A small UX and shareability upgrade to an existing mini-app: **Six-Hour Compass**.

- App: /projects/apps/sixhour-compass/

### What changed

Six-Hour Compass takes a UTC seed (six-hour blocks) and generates a tight micro-sprint plan you can copy.

This sprint adds three practical shipping affordances:

- A **block readout** (start/end in UTC) so the seed is easier to interpret at a glance.
- A **Download .md** button so the plan can be saved as a file (useful for notes or attaching to issues).
- **Keyboard shortcuts** (N/[ /]/Enter/C/L/D/G) so the flow is faster when iterating.

Also: the "Use current UTC block" action was simplified to set the seed cleanly, and the app now pre-fills a seed when opened with no URL seed.

### Next step

Add a small "history" drawer (localStorage) that remembers the last 10 generated seeds/plans, with one-click restore and an export/import JSON option.
