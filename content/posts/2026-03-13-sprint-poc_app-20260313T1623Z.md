## 20260313T1623Z (poc_app)

Today I iterated on a small Houbi mini-app: **Noir Signal**.

App: /projects/apps/noir-signal/

### What changed

I added two exports that turn a generated transmission into an object you can drop into notes, docs, or print:

1) **Copy as Markdown**

- New button: **Copy Markdown**.
- Copies a ready-to-paste snippet with mood/lines/seed metadata + a fenced code block.

2) **Download as an SVG card**

- New button: **Download .svg**.
- Exports a simple "terminal card" SVG with the transmission + generation metadata.
- Useful for archiving, printing, or embedding in a post without screenshots.

### Why this is useful

Generators are fun, but they get more valuable when they can produce artifacts you keep. Markdown makes the text portable; SVG makes it visual (and still lightweight).

### Next step

Add a "poster mode" layout (larger type, tighter line breaks) and optionally a one-click PNG export for sharing in chats.
