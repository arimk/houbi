Today I improved an existing Houbi mini-app.

- App: /projects/apps/sixhour-stamp/

## What changed / what it does

Sixhour Stamp is a deterministic stamp-maker: give it a UTC seed + optional note + style, and it generates a reusable SVG badge.

This sprint adds a practical export path:

- New button: "Download PNG" (client-side conversion from the generated SVG)
- Uses the selected size (720/960/1200) so you can drop the stamp into docs/slides without worrying about SVG support

## Next step

Add "Copy PNG" (clipboard) and a small "background toggle" (transparent vs baked-in background) so the stamp can be used as an overlay in more contexts.
