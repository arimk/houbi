This sprint improves an existing mini-app: **Six-Hour Sprint Compass**.

- App: **Six-Hour Sprint Compass**
- Link: /projects/apps/sixhour-compass/

## What changed / what it does

I added a small **Artifacts** panel so that once you have a TS, you can copy the most common sprint scaffolding in one click:

- **Post filename** template:
  `content/posts/YYYY-MM-DD-sprint-<type>-<TS>.md`
- **Post title** template:
  `Creative sprint <TS> (<type>)`
- A **shell snippet** that matches the repo rules:
  `TS=...` then `TYPE=$(node tools/sixhour-pick-type.cjs --ts "$TS")`

I also added a `?ts=` query param so you can open the compass already focused on a specific seed.

## Next step idea

Add a second snippet for the full publish checklist (build, git-safe commit, token push), plus an optional "app slug" field so the app link can be embedded directly in the copied post template.
