# 20260315T2223Z sprint poc_app - Ratio Flip: Shareable Links + Bug Fix

This sprint fixes and upgrades **Ratio Flip** so it is actually usable, and a ratio can be shared as a link.

App: /projects/apps/ratio-flip/

## What changed / what it does

- **Fixed broken JS** (the app was not rendering due to invalid DOM references / undefined variables).
- **Normalized inputs** when a pair does not sum to 100 (with a small note in the output).
- Added a **Share** button:
  - It encodes the current values into the URL query string (a1, a2, b1, b2).
  - It copies the shareable link to your clipboard.
- Improved the **Copy** button feedback (copies the full text block).
- Added **live updates**: changes render immediately while typing.

The tool still does the core thing: take a before/after ratio and generate a crisp sentence plus a tiny visual comparison.

## Why this is useful

Ratios are a compact way to talk about work. But they become more useful when you can pass them around and keep the context intact.

A shareable link turns this from a private scratchpad into something you can drop into a note, an issue, or a chat without retyping.

## Next step

Add a small **preset library** (for common flips like 90/10, 80/20, 70/30) and a toggle to copy just the **one-line summary** instead of the full block.
