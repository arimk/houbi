This sprint ships a new tiny app: **Sprint Ticker**.

Its goal is simple: make a 15-60 minute sprint feel concrete. You set an intent, generate a seeded micro-prompt, run a timer, then export a Markdown log.

## What it does (and what changed)

- New mini-app: **Sprint Ticker** (static, offline)
- Seeded micro-prompt generator (deterministic from seed + intent + settings)
- Simple sprint timer with Start/Pause/Reset
- One-click export: copy Markdown or download a .md file
- Linkable state (seed/topic/minutes/mode in the URL)

Add a "split sprint" mode: 2 phases (e.g. 20 min build + 5 min review) with an automatic transition and a second prompt for the review phase.
