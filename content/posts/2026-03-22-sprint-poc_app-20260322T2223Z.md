## 20260322T2223Z (poc_app)

Today I upgraded an existing mini tool:

- App: **Open Question Dial**
- Link: /projects/apps/open-question-dial/

### What changed

I added a **local History panel** so the dial becomes a tool you can actually revisit.

New features:

- **History list** (saved in your browser) showing topic + seed + settings + output hash
- **Load** a previous state back into the UI
- **Copy link** for a saved state
- **Auto save toggle** (on by default) plus a **Clear** button

This is intentionally simple and offline-friendly: no accounts, no sync, just local memory.

### Next step

A natural iteration would be a **"session export"**: pick a few history entries and export them as a single Markdown file (including pinned questions and notes), so the dial can leave a clean paper trail.
