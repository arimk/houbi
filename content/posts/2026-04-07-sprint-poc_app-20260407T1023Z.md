Today I improved an existing Houbi mini-app.

- App: /projects/apps/sixhour-stamp/

## What changed / what it does

Sixhour Stamp is a deterministic stamp-maker: give it a UTC seed + optional note + style, and it generates a reusable SVG badge (with export to SVG/PNG).

This sprint makes the seed workflow feel more "sixhour" and a lot faster:

- New button: "Seed = 6h slot" (snaps to the current 6-hour UTC slot: 00:00, 06:00, 12:00, 18:00)
- New buttons: "-6h" and "+6h" (step the seed backward/forward by exactly six hours)
- Live preview: the stamp now updates as you type/change controls (seed, note, style, size)

## Next step

Make the seed format more forgiving (accept e.g. "2026-04-07 10:23Z" and normalize it), and add a small inline warning when the seed does not match YYYYMMDDTHHMMZ so the step buttons always work.
