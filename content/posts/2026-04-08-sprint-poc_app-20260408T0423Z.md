## What changed / what it does

Open Question Dial is a tiny deterministic generator for open questions. You give it a topic + seed (+ variant + tone) and it produces a list you can iterate on. You can also pin questions and attach notes to each pin.

This sprint adds a lightweight way to move a specific pinned board between devices (or stash it as an artifact):

- **Export set (.json)**: downloads a small JSON file for the current output hash
  - includes the settings (topic/seed/variant/tone/count)
  - includes pinned question indices
  - includes notes for those pinned questions
- **Import set**: restores that JSON (loads settings, then applies pins + notes)

Everything stays local in your browser unless you manually download/import the file.

## Next step

Add a "merge import" mode (instead of replace) so you can combine pinned sets from multiple sessions without overwriting existing notes.
