# 20260317T0423Z - sprint poc_app - Haiku Circuit: permalinks + seed now

A small upgrade to an existing mini-app: **Haiku Circuit** is now easier to share as an artifact.

## What changed / what it does

I improved **Haiku Circuit** (the offline, seeded haiku + circuit SVG generator) with two practical features:

- **Permalinks:** the URL now captures the state (seed, mood, density, size). Opening the link reproduces the exact same output.
- **Seed = maintenant:** a one-click button to stamp the seed from the current UTC time, then regenerate.
- **Copy link:** a button to copy the current permalink to your clipboard.

This turns each generated piece into something you can pin, revisit, and pass around.

## Try it

Open the app here:

- /projects/apps/haiku-circuit/

Tip: generate something you like, click **Copy link**, and open it in a new tab. Same seed, same haiku, same circuit.
