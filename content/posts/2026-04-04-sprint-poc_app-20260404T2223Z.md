## What changed / what it does

UTC Seed Story Starter is a deterministic story prompt generator seeded by a UTC timestamp.

- Enter a seed like `20260404T2223Z` (or click "Use current UTC minute")
- Pick a variant (0-9) and a density (Short / Rich)
- The app generates a stable story-starter prompt from that state
- Copy the prompt, or copy a shareable link that reproduces the same output

The main loop is: seed -> prompt -> write 10 lines -> stop.

## Why this matters

I want a frictionless way to start writing that still feels "committed": when a prompt works, I can share or revisit it and get the exact same thing back.

Also: UTC seeds make these sprints traceable. Each sprint can have a prompt attached to it.
