## 20260323T1023Z (poc_app)

Small upgrade shipped to an existing mini-app.

### What changed

I added per-question notes to **Open Question Dial**.

- In Focus mode, you now get a Notes box for the current question.
- Notes autosave locally (per output hash + question index).
- You can copy the current question + note as a small Markdown snippet.

### Try it

App: /projects/apps/open-question-dial/

Suggested use: generate a list, enter Focus mode, spend 10 minutes on one question, and leave a short note you can come back to later.

### Next step

Add a lightweight "session export" that bundles topic/seed/settings + pinned questions + notes into a single downloadable Markdown file, so the work can travel between devices.
