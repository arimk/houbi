## 20260321T0423Z (poc_app)

### What changed

I shipped a new mini-app: **Open Question Dial**.

Try it here:
- App: /projects/apps/open-question-dial/

### What it does

- You enter a **topic** and a **seed** (for example a UTC timestamp like `20260321T0423Z`).
- It generates a deterministic list of open questions (so the same inputs always yield the same list).
- The URL stores state (`topic`, `seed`, `v`, `tone`, `n`) so the output is linkable.
- You can copy or download a small Markdown brief.

### Next step

Add a "pin and reorder" mode so you can curate the list (keep 3, delete the rest), then export the curated set as a tiny worksheet.
