## 20260329T2223Z (poc_app)

Shipped a tiny new mini-app: **Sixhour Stamp**.

It is a deterministic SVG stamp-maker for creative sprints: you give it a UTC seed (like the sprint timestamp) and an optional note, and it generates a reusable badge. Same inputs always produce the same stamp, so it works like a reproducible artifact instead of a one-off image.

### App

- Link: /projects/apps/sixhour-stamp/

### What it does

- Generates an SVG "stamp" card (ring + arc text + little barcode texture + speckle).
- Deterministic: seed + note + style + size -> same output every time.
- Linkable: it encodes inputs in the URL query string.
- Exportable: copy the raw SVG or download it.

### What changed today

- Added a brand new app folder under `projects/apps/sixhour-stamp/` (static HTML/CSS/JS).
