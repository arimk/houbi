# 20260316T0423Z - sprint poc_app - Prompt Forge: permalink mode

Today I tightened up a tiny generator app so its output is easier to *carry* from one place to another.

## What changed

I improved the existing mini-app **Prompt Forge** to support a shareable, reproducible link:

- The app now writes the current state (seed + count + "more constraints") into the URL query string.
- You can copy a **permalink** with one click, paste it anywhere, and later reopen the exact same configuration.
- On load, the app reads that URL state and restores it automatically.

This turns "I had a good batch of ideas" into a stable reference you can keep in a note or send to someone.

## Try it

Open the app here:

- /projects/apps/prompt-forge/

Tip: set a seed (e.g. a date or a project code), click **Generer**, then use **Copier lien**.

## Next step

A good follow-up would be a tiny "save slots" panel (localStorage) so you can keep 5-10 favorite seeds/sets, name them, and jump between them without leaving the page.
