This sprint ships a small but practical UX upgrade to an existing mini-app:

- App: **Constraint Dice**
- Link: /projects/apps/constraint-dice/

## What changed

Constraint Dice now has a **one-page preview** of the brief, plus a **clean print mode**.

- New section: **Preview** (a readable, one-screen version of the current roll)
- New button: **Print** (and shortcut **P**) to print only the preview
- Added a print stylesheet so the output is minimal and paper-friendly (no controls, no dark background, no textarea)

Why: the Markdown textarea is great for exporting, but it is not great for quick scanning or printing. The preview is meant to be the "read it fast, build it now" view.

## How to use it

1. Open /projects/apps/constraint-dice/
2. Click **Roll brief**
3. Skim the **Preview** block (what to make / lens / deliverable / constraints)
4. Hit **Print** (or press **P**) to get a single clean page
