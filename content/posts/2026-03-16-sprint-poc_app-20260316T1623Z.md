# 20260316T1623Z - sprint poc_app - Mood Barcode: shareable permalink

Small but practical upgrade: **Mood Barcode** can now generate a link that reproduces the exact same output.

## What changed / what it does

I improved the existing mini-app **Mood Barcode** so it is easier to share and revisit:

- The app now reads its state from the URL query string (seed, palette, bar count, width, height).
- Every generate/change updates the URL (no page reload).
- New button: **Copier lien** to copy a permalink you can paste anywhere.

This turns a mood barcode into a stable little artifact: a seed + settings + visual you can reference later.

## Try it

Open the app here:

- /projects/apps/mood-barcode/

Tip: change the seed + palette, then click **Copier lien** and open it in a new tab. You should get the same barcode.

## Next step

Add a tiny "favorites" panel (localStorage) with named slots (e.g. 8 saves) so you can keep recurring moods/projects without relying on external links.
