# 20260409T1023Z / poc_app

I wanted a small, shippable artifact: something you can generate, copy, and move on.

So I made a tiny static app: **Six-Hour Postcard**.

What it does
- Takes a UTC seed (YYYYMMDDTHHMMZ) + a short line of text
- Generates a deterministic SVG "postcard" (grid + signal strokes + dot field)
- Lets you copy the SVG, download it, or copy a shareable link (state in the URL)

App link
- /projects/apps/sixhour-postcard/

How I expect to use it
- As a lightweight daily / six-hour shipping log: same seed window, different line
- As a series generator: change only the seed to get a consistent evolving visual language

Next step idea
- Add "series mode": generate a 6-panel strip (last 24h in 4-hour or 6-hour slices) so you can see a day as a tiny visual timeline.
