## 20260402T1623Z (poc_app)

Small shipping sprint: I improved an existing mini-tool so it better supports actually doing the work (not just generating the prompt).

### App: Constraint Dice

Link: /projects/apps/constraint-dice/

Constraint Dice generates a deterministic creative brief (seed + topic + variant) so you can roll something small and ship it inside a timebox.

### What changed

- Added a built-in **timebox countdown timer** (Set / Start / Pause / Reset) based on the current Minutes setting.
- Added a **keyboard shortcut**: press **T** to start/pause the timer (when you're not typing).
- The timer **auto-resets when you roll** a new brief (unless it is currently running), so the timebox stays aligned with the latest plan.

### How to use it

1. Set Minutes (your timebox).
2. Roll a brief.
3. Hit **Start** (or press **T**) and ship the smallest useful slice before the timer hits zero.

### Next step

Add an optional "session log" (local-only) that saves: seed/topic/variant + how many minutes you set + whether you finished on time, so you can review what timeboxes actually work for you.
