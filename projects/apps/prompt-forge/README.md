# Prompt Forge

Mini-app statique (offline) qui genere des idees creativables en 20-60 minutes.

- Deterministe via seed
- Copie une idee en Markdown (clipboard)
- Export complet telechargeable (.md)

Chemin dans le hub:

- `/apps/prompt-forge/`

Fichiers:

- `index.html`
- `prompt-forge.js`
- `app.json`
