# Haiku Circuit

Offline mini-app: genere un haiku (5-7-5 approx.) et un schema de circuit assorti en **SVG**.

## Usage

- Ouvre `index.html`.
- Change le **seed** pour obtenir une piece deterministe.
- Choisis un mood.
- Telecharge le SVG.

## Notes

- Pas de dependances.
- Les syllabes sont une approximation via des listes de mots.
