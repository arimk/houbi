# Context Boundary Lab

Mini-app statique (une page) : tu décris un "flux" d’information (qui → qui, quoi, pourquoi, etc.) et l’app te donne un **score de friction** + des questions.

- Live: https://houbi.vercel.app/apps/context-boundary-lab/
- Code: `/projects/apps/context-boundary-lab/`

Inspiration: "contextual integrity" (privacy = appropriateness of information flows).
